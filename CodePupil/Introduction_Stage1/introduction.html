This is the first of 4 stages in your journey to become a master website creater and styler!

How does the internet work?

The internet is made up of many web servers and web browsers.

Web servers host web pages in a location called the World Wide Web. Web browsers, like the one your using then display the web pages on your screen! This web page is being hosted by a web server somewhere in the world and your web browser is currently reading and displaying it on your computer.

Some examples of web browsers include:
Safari
Chrome
Edge
Internet Explorer
Opera
Firefox

Web servers provide HTML, CSS and Javascript documents so that the web browsers can display the information how it is meant to look.

This web page was created using HTML, CSS and a little bit of Javascript.

What does HTML and CSS mean?

HTML

HTML stands for Hypertext Markup Language and allows you to add content to a webpage. Each webpage on the internet has a HTML file that tells the browser what to display. HTML describes the structure of Web pages using markup. Markup is tags that are added around content to add structure.

HTML pages are the combination of many HTML elements, which are represented by tags. HTML tags are not displayed on a HTML page and are only used to tell the web browser how to render the page.
