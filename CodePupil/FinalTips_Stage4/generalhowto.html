Some final tips before you head off!

Organising your code

Now if your like me and you don't like unorganised documents you probably want to do something about it. And if I told you that you couldn't then I would be lieing. To create a folder where you can store you documents all you have to do is right click inside the grey box just below your coding document.

Like the image below:

Now drag your html document into the folder and reload the page. You will find that the images will change to alt text and the links and external CSS won't work.

Don't panic. There is an easy fix to this. When the href attribute links to another document or image it can only see everything inside its folder.

To move out of a folder type ../ before the document name. This will now make the href look at the area outside this folder. You can repeat this or dive into a different folder.

To dive into a folder just follow this syntax and put the name of the folder and then a slash.

NAME-OF-FOLDER / image-name.png

This file organisation is really important if you have multiple web pages and lots of images.
